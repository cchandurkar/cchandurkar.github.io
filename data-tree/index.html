Data Tree

Data oriented tree structure that enlishes the power of callbacks to create, search and traverse tree.

Install:

$ npm install data-tree

Then use via require('data-tree') OR use browserified builds from dist/dataTree.js OR dist/dataTree.min.js in <script /> tag. Browserified builds are also available on github.io at dataTree.js and dataTree.min.js
